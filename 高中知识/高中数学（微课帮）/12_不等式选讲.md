## 1 绝对值不等式

### 解法

1. 形如 $|ax + b| \geq |cx + d|$ 的不等式，直接两边平方
2. 形如 $|ax + b| \leq c(c\gt0)$ 和 $ |ax + b|\geq c(c\gt0) $
    - $ |ax + b|\leq c \Longleftrightarrow -c \leq ax + b\leq c\,(c > 0)$
    - $ |ax + b|\geq c \Longleftrightarrow ax + b\geq c\quad or\quad ax + b\leq -c(c>0) $

### 应用

定理：如果 a, b 是实数，那么 $|a+b|\leq|a|+|b|$, 当且仅当 $ab\geq 0$ 时，等号成立。

推论1：如果 a, b, c 是实数，那么 $|a-c|\leq|a-b|+|b-c|$, 当且仅当 $(a-b)(b-c)\geq 0$ 时，等号成立（即把定理中的 a 换成 a-b，把 b 换成 b-c）

推论2：$|a_1 + a_2 + ... + a_n|\leq|a_1| + |a_2| + ... + |a_n|$

推论3：$||a| - |b||\leq|a+b|\leq|a| + |b|$

推论4：$||a| - |b||\leq |a-b|\leq |a| + |b|$ （即把推论 3 中的 b 换成 -b）

`Example` 若存在实数 x 使 $|x-a|+|x-1|\leq3$ 成立，则实数 a 的取值范围是 ___



`Solution`

存在一个数，要使左边比右边小，则只要左边的最小值 <= 3 就行。

左边最小值为 $|x-a|+|x-1| \geq |x-a-(x-a)| = |a-1|$

解得 $a \in [-2, 4]$
